# Digital_Twin_Working_Demo

This repository consists of four python files which step-wise-step have a demo working of Digital Twin to show a simple demonstration of getting temperature, occupancy and power of a room which can be later turned into analytics for further decision making
There are 4 files - simulator, publisher, subscriber and the main app file to test this software
To ideally have the results on a monitor, code for simple webpage is also added

How to Run (IMPORTANT):

Run MQTT publisher

Run MQTT subscriber (see data flowing)

Run app.py 

Open page.html in browser
